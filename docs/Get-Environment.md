# Get-Environment

## SYNOPSIS
Get system environment variables

## SYNTAX

```
Get-Environment [[-Name] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Returns system environment variables

## EXAMPLES

### EXAMPLE 1
```
Get-Environment
```
