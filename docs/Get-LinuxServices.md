# Get-LinuxServices

## SYNOPSIS
Get Linux services

## SYNTAX

```
Get-LinuxServices [[-Name] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Get Linux services and basic configuration settings

## EXAMPLES

### EXAMPLE 1
```
Get-LinuxServices
```

### EXAMPLE 2
```
Get-LinuxServices -Name "foo"
```
