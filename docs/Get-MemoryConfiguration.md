# Get-MemoryConfiguration

## SYNOPSIS
Get memory configuration.

## SYNTAX

```
Get-MemoryConfiguration
```

## DESCRIPTION
Get memory configuration.
This function uses dmidecode to get memory configuration.

## EXAMPLES

### EXAMPLE 1
```
Get-MemoryConfiguration
Get memory configuration for each bank on the current chassis.
```
