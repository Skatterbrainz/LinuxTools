# Get-LocalUser

## SYNOPSIS
Get local users

## SYNTAX

```
Get-LocalUser [[-Identity] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Get local user accounts

## EXAMPLES

### EXAMPLE 1
```
Get-LocalUser
```

### EXAMPLE 2
```
Get-LocalUser -Identity "frank.zappa"
```
