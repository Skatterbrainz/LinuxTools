# Update-LinuxPackages

## SYNOPSIS

Update and upgrade Linux packages

## SYNTAX

### __AllParameterSets

```
Update-LinuxPackages [[-Type] <string>] [<CommonParameters>]
```

## DESCRIPTION

Update and upgrade Linux packages

## EXAMPLES

### EXAMPLE 1

Update-LinuxPackages

Updates all packages

### EXAMPLE 2

Update-LinuxPackages -Type "apt"

Updates only apt packages
