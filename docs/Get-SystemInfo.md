# Get-SystemInfo

## SYNOPSIS
Gets detailed system information from a Linux system.

## SYNTAX

```
Get-SystemInfo [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Collects information about CPU, memory, disk, and operating system.

## EXAMPLES

### EXAMPLE 1
```
Get-SystemInfo
```
