# Test-RebootPending

## SYNOPSIS
Test if a reboot is pending.

## SYNTAX

```
Test-RebootPending [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Test if a reboot is pending.
This function checks for a reboot-required file on Debian-based systems.
