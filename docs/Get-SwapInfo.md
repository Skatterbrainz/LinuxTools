# Get-SwapInfo

## SYNOPSIS

Gets Linux swap configuration and usage.

## SYNTAX

### __AllParameterSets

```
Get-SwapInfo [<CommonParameters>]
```

## DESCRIPTION

Returns swap devices/files, usage, and vm.swappiness.

## EXAMPLES

### EXAMPLE 1

Get-SwapInfo

Returns swap usage and swappiness settings.
