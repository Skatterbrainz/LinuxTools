# Get-GPUInfo

## SYNOPSIS

Gets GPU adapter and driver information.

## SYNTAX

### __AllParameterSets

```
Get-GPUInfo [<CommonParameters>]
```

## DESCRIPTION

Returns GPU adapter details from `lspci` and NVIDIA runtime details from `nvidia-smi` when available.

## EXAMPLES

### EXAMPLE 1

Get-GPUInfo

Returns GPU adapter and driver information.
