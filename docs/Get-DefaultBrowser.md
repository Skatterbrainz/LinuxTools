# Get-DefaultBrowser

## SYNOPSIS
Get the default browser.

## SYNTAX

```
Get-DefaultBrowser
```

## DESCRIPTION
This function reads the ~/.config/mimeapps.list file to determine the default browser.

## EXAMPLES

### EXAMPLE 1
```
Get-DefaultBrowser
Returns the default browser.
```
